## 1. 评分架构总览

```
Final = L3(0-30) + L4(0-45) + L5(0-25) = 100pt
Pool: ≥75 核心池 | 50-74 观察池 | <50 备选池
```

所有 L3/L4/L5 得分由 **Python 规则引擎** 计算，不是分析 Agent 编的。

## 3. L4 穿透回报率 (Penetration Return, 0-45pt)

### 核心公式
```
PR = (可支配现金 × 分配比率 + 回购注销) / 当前市值
```

### 各组件释义

**可支配现金** (Disposable Cash) =
`经营CF净额 - 维持性CAPEX - 并购子公司 - max(0,年末长投-年初长投) - 财务费用 + 货币资金 - 限制性货币 - 短期借款 + 交易性金融资产`

- 维持性CAPEX = CAPEX × maintenance_coefficient（行业先验值 + 资产密度加权，见配置文件）
- 数据均取最近披露年度数据

**分配比率** (Distribution Ratio)，按优先级：
1. **公告分红承诺** (tier1) — 从 Web 搜索获取，原值直接使用
2. **历史外推** (tier2) — mean(近5年 年度分红总额 / 年度净利润)

**回购注销** — Web 搜索 + LLM 提取，仅计入「用于注销」的回购金额

### 打分映射

| PR 范围 | 起点分 | 标签 |
|---------|--------|------|
| ≥12% | 20 | 优秀 |
| ≥8% | 15 | 良好 |
| ≥5% | 10 | 及格 |
| <5% | 0 (L4=0) | 不及格 |

L4 = max(起点分 - OE质量扣分, 0)，上限45

### OE 质量标签（影响 L4 折扣）

OE_cf = 经营CF - 维持性CAPEX（Python 计算，5年中位数）

- **🟢可信**: OE_cf/净利润 ≥ 0.8 AND OE_cf CV ≤ 0.3 → L4 正常
- **🟡存疑**: (0.5 ≤ OE_cf/净利润 < 0.8) OR (0.3 < OE_cf CV ≤ 0.5) → L4 × 0.7
- **🔴不可靠**: OE_cf/净利润 < 0.5 OR OE_cf CV > 0.5 → L4 = 0

## 4. L5 安全边际 (0-25pt)

### 4.1 估值安全边际率 (0-15分)

```
合理市值 = (DC × 分配比率 + 回购注销) / 7%
安全边际率 = (合理市值 - 当前市值) / 当前市值
```

折现率 7% = max(无风险利率+2%, 5%) + 个股风险溢价 2%

| 安全边际率 | 得分 |
|-----------|------|
| ≥50% | 15 |
| ≥30% | 12 |
| ≥15% | 8 |
| ≥0% | 4 |
| <0% | 0 |

### 4.2 下行风险缓冲 (0-5分)

- **资产底价** (0-2分): (货币资金+交易金融资产-总负债)/市值, ≥50%=2, ≥20%=1
- **股息托底** (0-1分): 当前股息率 ≥ 历史中位数×0.8 → 1分
- **回购支撑** (0-2分): 有回购注销 → 2分

### 4.3 仓位 (0-5分, 安全边际率单维驱动)

| 安全边际率 | 仓位 | 得分 |
|-----------|------|------|
| ≥30% | 15% | 5 |
| ≥15% | 10% | 3 |
| ≥0% | 5% | 2 |
| <0% | 0% | 0 |

## 5. CV Agent 核查规则

### 5.1 什么由管线算 vs 什么由分析 Agent 定性

| 数值/标签 | 来源 | CV 怎么办 |
|-----------|------|----------|
| L3/L4/L5 得分 | Python 规则引擎 | **只核对数值一致性**（分析报告有没有抄错），不需要在 brief.md 里搜是否存在 |
| PR / 安全边际率 | Python 公式 | 用 brief.md 原始数据 + 已知公式 **复算验证** |
| OE 标签 (🟢/🟡/🔴) | Python 规则引擎 | 核对分析报告是否正确引用了标签 |
| 9模块定性分 (护城河 5/5 等) | 分析 Agent 主观 | 核查 **证据链是否引用 brief.md 数据**，不评判分数是否「合理」 |
| 商业知识文字 | LLM 搜索 | 核查 **是否与 brief.md Section 5 一致** |

### 5.2 常见误判避免

- **看到 "穿透回报率 4.33%" 在 brief.md 里搜不到 4.33 → 不要标 ?缺乏证据**
  4.33% 是从 brief.md 中的 DC/分配比率/市值三个数据按公式算出来的中间值，管线代码完全正确。CV 应该用公式复算验证。
- **看到 "综合评分 37/45" → 不要标 ?缺乏证据**
  这是分析 Agent 的 9 模块 qualitative_score，与管线的 Final=100pt 不同体系，不是数据引用错误。
- **看到 OE 标签 → 只需核对是否与 brief.md 一致，不需要判断标签合理性**

### 5.3 严格性边界

- **数值引用**: 必须精确一致（允许 ±0.1% 四舍五入）
- **趋势判断**: brief.md 财报洞察 Section 三 中有明确支持 → ✓
- **商业断言**: brief.md Section 五 中有对应内容 → ✓
- **管线计算**: 用公式复算（DC × ratio + buyback）/ market_cap → 与报告数值匹配 → ✓
- **分析 Agent 自评分数**: 仅核查引用的数据是否有误，不评判分数合理性
