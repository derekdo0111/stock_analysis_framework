# 2026-06-07 — v0.35 多策略架构重构

## 重构目标
将单体龟龟策略项目重构为多策略插件架构，为后续 Hermes Agent 云端维护和新策略扩展做准备。

## 新架构
```
src/
├── core/                    ← 🟢 所有策略共用
│   ├── data/               ← 合并 data_fetcher + data_pool
│   ├── llm/                ← 5 个通用文件
│   └── utils/
├── turtle/                  ← 🔴 龟龟策略完整内聚
│   ├── cli.py              ← stock-analyze 入口
│   ├── calculator/         ← L2-L5 + financial_deep_analysis + financial_ratios + unit_converter
│   ├── screening/          ← hard_gate + classifier + l2_screener + run_screener
│   ├── llm/                ← 8 个龟龟专属 Agent
│   ├── reporter/           ← brief.md + HTML
│   └── rules/              ← YAML + Pydantic schema + loader/injector/validator
├── strategies/              ← 🔵 策略插件层
│   ├── base.py             ← BaseStrategy ABC
│   ├── registry.py         ← 自动发现
│   └── turtle.py           ← 龟龟适配器
├── agents/                  ← 不变
└── backtest/                ← 不变
```

## 通用 vs 专属判定
- core/llm/: client/cache/tools/provider/manager (零策略依赖 ✓)
- turtle/llm/: 8 个 Agent 全部依赖 FinalScore → 乌龟专属
- turtle/rules/: loader/injector/validator 全部 import turtle schemas → 从 core 撤回
- 已删除: src/llm/schema.py (死代码, 0 引用)
