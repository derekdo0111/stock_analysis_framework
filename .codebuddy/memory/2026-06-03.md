## L3 十二维商业模式评估 + L5 估值安全边际重构 + 加法百分制

### 核心变更
- **评分公式**: `Final = (L2+L4+L5)×L3` → `Final = L3_30pt + L4_45pt + L5_25pt = 100pt`
- **L3 从乘法器变加法**: 旧 `×1.2/×1.0/×0.8/reject` → 十二维 0-30pt 加法打分
- **L5 重构**: 去掉外推可行度6维+价值陷阱5项, 改为纯估值安全边际(估值安全边际率+下行缓冲+仓位)
- **L2 降级**: 纯门控, 不参与最终评分
- **折现率**: 7% = max(无风险利率+2%, 5%) + 个股风险溢价2%
