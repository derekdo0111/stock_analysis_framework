## PR 公式根本性范式转换讨论 (11:30-12:00)

### 用户发现
- **旧公式**：`PR = (分红+回购) / 年末总市值` — 分母是市场定价，股价跌PR翻倍，本质是"市场先生的函数"
- **新公式**：`PR = (分红+回购) / 真实可支配现金` — 分母是公司口袋里真正能花的钱

### 真实可支配现金公式（用户定义）
```
可支配现金 = 经营CF净额(n_cashflow_act)
           - 投资活动现金流出小计(stot_out_inv_act)  ← 所有投资支出全扣，成长性投入视为打水漂
           - 财务费用(fin_expense)
           + 货币资金(money_cap)
           - 限制性货币  ← ❌ Tushare无此字段
           - 短期借款(st_borrow)
           + 交易性金融资产(tradable_fin_assets)  ← 短期理财视为现金等价物
```

### 关键设计决策
- **CAPEX vs 全部投资支出**: 用户纠正——不是仅减CAPEX，而是减去所有投资活动现金流出（建厂+买设备+参股+并购），全部视为成长赌注可能打水漂
- **短期理财的处理**: 买短期理财在"投资流出"中扣掉，但通过末尾"+交易性金融资产"加回，净效果为零（现金→理财→仍是可支配的）
- **参股/并购**: 进入长期股权投资而非交易性金融资产，不被加回，永久扣掉 ✅
- **时间错配固有限制**: 年初买年末到期的理财会被误扣（现金流是全年累计，BS是年末快照），但年末money_cap部分对冲
